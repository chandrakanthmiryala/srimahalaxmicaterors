# SRI MAHALAXMI CATERERS
## Dynamic Website Project Documentation

---

## 1. Project Overview

**Website Name:** SRI MAHALAXMI CATERERS  
**Type:** Dynamic Catering Business Website  
**Purpose:** To showcase catering services, menu items, gallery, customer reviews, and allow admin to manage content dynamically and
menu custamization.

---

## 2. Website Goals

- Showcase catering services and menu items
- Display high-quality food images and videos
- Allow admin to upload and manage content
- Enable editing of landing page content
- Collect and display customer reviews
- Provide easy contact options for customers
- Modern, attractive, and mobile-friendly design

---

## 3. Website Structure (Frontend Pages)

### 3.1 Landing Page (Home)
- Hero banner with slideshow (images/videos)
- Business name and tagline
- Quick introduction about the catering service
- Featured menu items
- Call-to-action buttons (Book Now / Contact Us)
- Highlights (Experience, Quality, Hygiene, Events Covered)

Editable by Admin:
- Banner images/videos
- Heading & tagline
- About section text
- Featured items

---

### 3.2 About Us Page
- Company history
- Mission & Vision
- Specialties
- Service areas
- Team information (optional)

---

### 3.3 Services Page
- Wedding Catering
- Birthday Parties
- Corporate Events
- Food Donation Services
- Outdoor Catering
- Custom Menu Packages

Each service should have:
- Description
- Images
- Price (optional)

---

### 3.4 Gallery Page
- Image gallery (grid layout)
- Video gallery
- Category filter (Weddings, Events, Food Items, Decorations)
- Lightbox preview

Admin Options:
- Upload images
- Upload videos
- Delete/Edit media
- Categorize gallery items

---

### 3.5 Menu Page
- Starters
- Main Course
- Rice Items
- Curries
- Sweets
- Special Packages

Admin Options:
- Add/Edit/Delete menu items
- Add price
- Add description
- Add item image

---

### 3.6 Customer Reviews Page
- Display customer testimonials
- Star rating system
- Customer name & event type
- Date of event

Admin Options:
- Approve/Reject reviews
- Delete reviews

Optional Feature:
- Customers can submit reviews via form

---

### 3.7 Contact Page
- Contact Form (Name, Phone, Email, Message)
- Phone numbers display
- WhatsApp button
- Google Map integration
- Social Media links

Admin Options:
- Edit phone number
- Edit email
- Update address

---

## 4. Admin Panel Features

### 4.1 Secure Login
- Username & Password
- Forgot password option

### 4.2 Dashboard
- Total inquiries count
- Total gallery images
- Total reviews
- Quick edit options

### 4.3 Content Management
- Edit Landing Page content
- Upload Images (JPG, PNG)
- Upload Videos (MP4)
- Manage menu items
- Manage services
- Manage testimonials

### 4.4 Media Management
- Image compression option
- File size limit control
- Delete or replace media

### 4.5 Contact Management
- View customer inquiries
- Export inquiries
- Delete old inquiries
